enum xnvme_cli_opt_type#

XNVME_CLI_LFLG: .

/cli &#8212;verbose; always optional XNVME_CLI_LOPT: ./cli &#8212;arg 0x0; optionally provide a value XNVME_CLI_LREQ: ./cli &#8212;arg 0x0; require providing a value XNVME_CLI_POSA: ./cli arg1 arg2; required, ordered, and provides a value XNVME_CLI_SKIP: ; This argument is used for formatting etc.
